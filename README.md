Tic-Tac-Toe Game
Overview
This is a simple implementation of the classic Tic-Tac-Toe game in Python. The game allows two players to play against each other in the terminal. The players take turns marking a space in a 3Ã—3 grid with their respective symbols (X and O). The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

Features
Two-player mode
Interactive command-line interface
Simple and easy-to-understand code structure
Validates moves and checks for wins/draws
Requirements
Python 3.x
